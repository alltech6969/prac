Pr2

A. Cryptool for Encryption and Decryption

 

            Steps:

                        1. Open Cryptool

                        2. You will see a .txt file delete everything in it.

                        3. Now write some text in it like "Hello World"

                        4. Go to Encrypt/Decrypt > Symmetric(modern) > RC4

                        5. Keep the key length 8 bit; change 00 to something like "12"

                        6. Click on Encrypt. (Encrypted Text is now visible)

                        7. Again go to Encrypt/Decrypt > Symmetric(modern) > RC4

                        8. Keep the key length 8 bit; and change 00 by "12"

                        9. Click on Decrypt. (Decrypted text is now visible).

                        5. right click on the wordlist file and Remove it

                        6. Again right click > add to list > wordlist.txt

Pr4

A. Using namp to perform -ACK, SYN, FIN, NULL, XMAS

            Steps:

                        1. 1. Launch Kali Linux

                        2. Open Terminal

                        3. Enter the following commands

                                    a. ACK Scan: sudo nmap -sA -T4 www.ruparel.edu

                                    b. SYN Scan: sudo nmap -sS -T4 www.ruparel.edu

                                    c. FIN Scan: sudo nmap -sF -T4 www.ruparel.edu

                                    d. NULL Scan: sudo nmap -sN -T4 www.ruparel.edu

                                    e. XMAS Scan: sudo nmap -sX -T4 www.ruparel.edu
